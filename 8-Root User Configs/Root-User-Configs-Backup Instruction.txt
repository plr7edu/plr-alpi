1. Go Root-User-Configs-Backup Directory

2.Copy Root-User-Configs-Backup folder ~/Desktop/ (Root User Account)

3.Run "1.BACKUP-ORIGINAL-CONFIGS Script" (First Time)
It creates root-original-user-config-backup directory in your home directory

4. Run "2.RESTORE-CONFIGS" Script 
This remove current configs and copy backup config files

5. "3.BACKUP-CONFIGS-NOW" Script can use to backup currently using settings
It Creates "Backup_time=+date".txt in Root User Configs Backup
